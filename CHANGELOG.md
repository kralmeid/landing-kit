# Changelog

All notable changes to this project are documented here.

## 0.1.0 - 2026-05-23

- Initial bilingual landing-page starter built with Next.js 16, TypeScript, Tailwind CSS, and next-intl.
- Added config-driven brand, navigation, SEO, imagery, portfolio, trust, and contact-form settings.
- Added Portuguese and English message files with locale-prefixed routes.
- Added an engineering consultancy preset under `examples/engineering/`.
- Added CI verification for typecheck, lint, and production build.
- Added Dependabot updates for npm dependencies and GitHub Actions.
